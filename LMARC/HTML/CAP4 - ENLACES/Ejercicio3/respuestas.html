RESPUESTAS

¿Qué es la informática?

Es la ciencia que estudia el tratamiento automático de la información mediante computadoras u ordenadores.

¿Qué es un sistema operativo?

Es el software principal que permite la comunicación entre el hardware y los programas de una computadora (por ejemplo, Windows, Linux o macOS).

¿Qué es el hardware?

Son los componentes físicos de un sistema informático, como el teclado, monitor, procesador, disco duro, etc.

¿Qué es el software?

Son los programas y sistemas que le indican al hardware qué hacer, como los navegadores, juegos, sistemas operativos, etc.

¿Qué es una red informática?

Es un conjunto de computadoras y dispositivos conectados para compartir información y recursos.

¿Cuál es la diferencia entre RAM y ROM?

La RAM es memoria de acceso aleatorio que se borra al apagar el equipo. La ROM es memoria de solo lectura que contiene información permanente.

¿Qué es una IP?

Es una dirección numérica única que identifica a un dispositivo en una red.

¿Qué es una base de datos?

Es un conjunto organizado de datos que pueden ser accedidos, gestionados y actualizados fácilmente.

¿Qué es un virus informático?

Es un programa malicioso diseñado para alterar el funcionamiento de un equipo sin el permiso del usuario.

¿Qué es la nube (cloud computing)?

Es un modelo de servicios que permite el acceso remoto a software, almacenamiento y procesamiento a través de internet.

¿Qué es un algoritmo?

Es un conjunto de pasos lógicos y ordenados para resolver un problema o realizar una tarea.

¿Qué es la virtualización?

Es la creación de una versión virtual de un sistema operativo, servidor, almacenamiento u otro recurso de red.
