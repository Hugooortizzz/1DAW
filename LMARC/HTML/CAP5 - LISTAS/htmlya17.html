Definiciones usadas en Informática

Algoritmo
Conjunto ordenado de instrucciones o pasos que permiten resolver un problema o realizar una tarea específica de manera sistemática
Base de datos
Conjunto estructurado de información o datos almacenados electrónicamente, que pueden ser gestionados y consultados mediante un sistema de gestión de bases de datos (SGBD).
Hardware
Conjunto de componentes físicos y tangibles de un sistema informático, como el procesador, la memoria, el disco duro o los periféricos.
Software
Conjunto de programas, aplicaciones y sistemas operativos que permiten que el hardware funcione y ejecuten tareas específicas para el usuario.
